# Einfach Digital Lernen

Klickbarer Prototyp einer Schulungsplattform fuer Menschen ab 65 Jahren.

## Starten

Die Website ist statisch und kann direkt im Browser geoeffnet werden:

```text
senioren-lernplattform/index.html
```

Optional kann ein lokaler Webserver im Ordner gestartet werden.

## Enthaltene Bereiche

- Startseite
- Onboarding mit Lernpfad-Vorschlag
- Dashboard
- Lernbereich und Moduldetail
- Simulationen
- Online-Banking-Simulation
- Bahnbuchungs-Simulation
- simulierte Website-Einbindung mit Warnhinweis
- Forum
- Chat / KI-Assistent
- Events
- Nutzerprofil

Alle Inhalte nutzen Dummy-Daten. Es werden keine echten Bank-, Ticket- oder Nutzerdaten uebertragen.
